# Big-O 
Jeff Elkner *January 88, 2022*

---

## Definition 

f(n) = O(g(n))

means there are positive constants c and k, such that 0 ≤ f(n) ≤ cg(n) for all
n ≥ k.

We say f(n) is *on the order of* g(n), or f(n) is *big O* of g(n).

---

## Example 

Show

`\[\begin{aligned}
4n^2 + 42n + 512 = O(n^2)
\end{aligned} \]`
